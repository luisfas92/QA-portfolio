# Nalu School - QA Project

## 1. Project Overview

Nalu School is a web-based e-commerce and booking platform focused on aquatic sports such as Surf, Kitesurf, and Skateboarding.

This QA project was developed to validate functional requirements, business rules, and core user flows including:

- User registration & authentication
- Product catalog navigation
- Shopping cart & checkout process
- Class booking system
- Public content modules (News, Brands, School)
- Responsive behavior

---

## 2. Scope of Testing

The following modules were tested:

- Homepage (Carousel & Navigation)
- Surf / Kitesurf / Skate categories
- Product detail pages
- Shopping cart
- Checkout process
- User login & registration
- Profile management
- Class reservations
- News & public sections

---

## 3. Test Strategy

Testing was performed using:

- Functional Testing
- Negative Testing
- Business Rule Validation
- Access Control Validation
- Cross-browser testing
- E-commerce flow validation

Browsers tested:
- Chrome

Operating Systems:
- Windows 11
- Windows 10
