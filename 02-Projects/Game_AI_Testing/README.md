# Game Overlay & AI Integration Android Testing 

## Project Overview

This project focuses on functional and light performance validation of a mobile overlay system (Entrypoint tab) integrated with Gemini Live AI across three Android games.

The goal was to ensure correct rendering behavior, gesture stability, AI activation, and performance impact during real gameplay scenarios.

## Objectives

- Validate Entrypoint visibility on game launch
- Verify overlay gesture behavior and UX stability
- Ensure proper widget alignment
- Test Gemini Live activation and response accuracy
- Evaluate gameplay performance with AI active (2-minute stress test)

## Games Tested

- The Battle of Polytopia  
- Gardenscapes  
- Angry Birds 2  

## Test Coverage

A total of 12 structured functional test cases were designed and executed:

Game, Rendering, Overlay, Gemini Activation and Performance.
